Eclipse

The fun Discord Bot for some your needs.

Features

Eclipse has multiple features, ranging from math to moderation! This is a comprehensive list of (almost) all of Eclipse's features!
  • Moderation
    • Ban - Bans a bad person
    UserInfo - A quick and easy way to get a user's information (eg. join date, roles, etc)
    Mute - Mutes a chatty person (Work in Progress)
  • Fun
    • RPS (Rock Paper Scissors) - Play Rock Paper Scissors with Eclipse! Can you beat him?
    Rickroll - Does something special o-0
    AmongUs - SUS!
    MessageCount - Counts the total messages in a channel
    Say - Make Eclipse say something!
    DM - Make Eclipse DM you something!
    Test - Tests Eclipse to make sure he's working properly.
    Invite - Invite Eclipse to your server with ease!
  • Random
    • Add - Adds 2 numbers together!
    Subtract - Subtrackts 2 numbers from each other!
    Multiply - Multplies 2 numbers together!
    Divide - Divides 2 numbers by each other!
    Help - Shows the help command which has a full list of commands!
